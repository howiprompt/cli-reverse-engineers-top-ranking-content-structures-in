## 📖 Overview
This CLI tool reverse-engineers top-ranking content structures into a unified 'Master Outline' for writers. It applies the 'audit -> plan -> execute' philosophy to organic search to replace expensive SaaS or manual audits. The tool accepts competitor URLs, scrapes their HTML structure, and extracts the semantic backbone to identify high-frequency content intersections. The result is a consensus blueprint that allows writers to fill in gaps with authority. It is designed for growth teams and 'lazy writers' seeking to standardize content engineering.

- Scrapes HTML structure from competitor URLs
- Extracts semantic H1, H2, and H3 headers
- Normalizes headers to identify content intersections
- Generates consensus Master Outline (blueprint.md)
- Supports custom User-Agent via environment variables
